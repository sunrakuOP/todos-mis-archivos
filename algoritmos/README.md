# 🧩 Algoritmos y Estructuras de Datos

Apuntes, implementaciones y retos de algoritmos y estructuras de datos.

---

## 📋 Temario

### 01 - Algoritmos de ordenación (Sorting)
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Comparativa: complejidad temporal y espacial (notación Big O)

### 02 - Algoritmos de búsqueda (Searching)
- Búsqueda lineal (Linear Search) — O(n)
- Búsqueda binaria (Binary Search) — O(log n)
- Aplicaciones prácticas

### 03 - Listas, Pilas y Colas
- Lista enlazada simple (Linked List)
- Lista doblemente enlazada
- Pila (Stack): LIFO — `push`, `pop`, `peek`
- Cola (Queue): FIFO — `enqueue`, `dequeue`

### 04 - Árboles y Grafos
- Árbol binario, árbol binario de búsqueda (BST)
- Recorridos: In-order, Pre-order, Post-order
- Grafos: representación con lista de adyacencia y matriz
- BFS (Breadth-First Search)
- DFS (Depth-First Search)

## 📊 Tabla de complejidades (Big O)

| Algoritmo | Mejor caso | Caso promedio | Peor caso |
|---|---|---|---|
| Bubble Sort | O(n) | O(n²) | O(n²) |
| Selection Sort | O(n²) | O(n²) | O(n²) |
| Insertion Sort | O(n) | O(n²) | O(n²) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) |
| Búsqueda binaria | O(1) | O(log n) | O(log n) |
