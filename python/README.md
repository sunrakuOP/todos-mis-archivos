# 🐍 Python

Apuntes, ejercicios y proyectos de Python 3.

## 📋 Temario

### 01 - Fundamentos
- Variables, tipos de datos (`int`, `float`, `str`, `bool`)
- Operadores aritméticos, de comparación y lógicos
- Entrada/salida: `input()`, `print()`
- Condicionales: `if`, `elif`, `else`
- Bucles: `for`, `while`, `break`, `continue`
- Funciones: definición, parámetros, retorno, `*args`, `**kwargs`

### 02 - Estructuras de datos
- Listas: indexación, slicing, métodos
- Tuplas: inmutabilidad, uso práctico
- Diccionarios: clave-valor, métodos
- Conjuntos (`set`): operaciones de conjuntos
- Comprensión de listas y diccionarios

### 03 - Programación Orientada a Objetos (POO)
- Clases y objetos
- Atributos y métodos
- `__init__`, `__str__`, `__repr__`
- Herencia y polimorfismo
- Encapsulación

### 04 - Módulos y paquetes
- Importar módulos estándar (`os`, `sys`, `math`, `random`, `datetime`)
- Crear y organizar módulos propios
- Uso de `pip` e instalación de paquetes externos
