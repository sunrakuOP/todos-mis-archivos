# 🎨 HTML + CSS

Apuntes, ejercicios y proyectos de maquetación web con HTML5 y CSS3.

---

## 📋 Temario

### 01 - HTML Básico
- Estructura de un documento HTML: `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`
- Etiquetas de texto: `<h1>`–`<h6>`, `<p>`, `<span>`, `<strong>`, `<em>`
- Listas: `<ul>`, `<ol>`, `<li>`
- Imágenes: `<img>` con `alt` y `src`
- Hipervínculos: `<a href>`
- Tablas: `<table>`, `<tr>`, `<th>`, `<td>`
- Formularios: `<form>`, `<input>`, `<textarea>`, `<button>`, `<select>`
- HTML semántico: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`

### 02 - CSS Básico
- Selectores: de etiqueta, clase (`.`), ID (`#`), atributo, pseudo-clases
- Modelo de caja (Box Model): `margin`, `border`, `padding`, `content`
- Colores, fuentes, fondos
- `display`: `block`, `inline`, `inline-block`, `none`
- `position`: `static`, `relative`, `absolute`, `fixed`, `sticky`

### 03 - Flexbox y Grid
- Flexbox: `flex-direction`, `justify-content`, `align-items`, `flex-wrap`
- CSS Grid: `grid-template-columns`, `grid-template-rows`, `gap`, `grid-area`
- Cuándo usar Flexbox vs Grid

### 04 - Diseño Responsive
- Media queries: `@media`
- Unidades relativas: `%`, `em`, `rem`, `vw`, `vh`
- Enfoque Mobile First
- `<meta name="viewport">`
