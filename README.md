# 📁 Todos mis archivos de programación

Repositorio personal para organizar todos mis apuntes, ejercicios y proyectos de programación.

---

## 🗂️ Estructura del repositorio

```
todos-mis-archivos/
├── python/          → Ejercicios y proyectos en Python
├── javascript/      → Ejercicios y proyectos en JavaScript
├── html-css/        → Maquetación web con HTML y CSS
├── java/            → Ejercicios y proyectos en Java
├── sql/             → Consultas y bases de datos SQL
├── algoritmos/      → Algoritmos, estructuras de datos y retos
└── proyectos/       → Proyectos transversales y finales
```

---

## 📌 Convención de nombres

Cada carpeta de lenguaje o tema sigue esta estructura interna:

```
lenguaje/
├── README.md            → Temario, ideas y recursos del lenguaje
├── 01-fundamentos/      → Conceptos básicos
├── 02-<tema>/           → Temas avanzados numerados
├── ejercicios/          → Ejercicios sueltos
└── proyectos/           → Mini-proyectos del lenguaje
```

Cada archivo de ejercicio usa este formato de nombre:

```
ejercicio_<numero>_<descripcion>.ext
```

---

## ✅ Plantilla para nuevos ejercicios

Copia `_plantilla_ejercicio.*` de cada carpeta para empezar un ejercicio nuevo.

## 💡 Cómo usar este repositorio

1. Entra a la carpeta del tema que estés estudiando.
2. Lee el `README.md` de esa carpeta para ver el temario y ejercicios sugeridos.
3. Trabaja los ejercicios en orden numérico.
4. Cuando termines un tema, pasa al siguiente número de carpeta.
5. Cuando tengas suficiente base, realiza un mini-proyecto en `proyectos/`.
