# 🗄️ SQL

Apuntes, ejercicios y proyectos de bases de datos con SQL.

---

## 📋 Temario

### 01 - Consultas básicas (DML)
- `SELECT`, `FROM`, `WHERE`
- Filtros: operadores de comparación, `AND`, `OR`, `NOT`
- Ordenar: `ORDER BY ASC/DESC`
- Limitar resultados: `LIMIT`
- `INSERT INTO`, `UPDATE`, `DELETE`
- Crear tablas: `CREATE TABLE`, tipos de datos (`INT`, `VARCHAR`, `DATE`, etc.)
- Restricciones: `PRIMARY KEY`, `NOT NULL`, `UNIQUE`, `DEFAULT`

### 02 - JOINs (combinar tablas)
- `INNER JOIN`
- `LEFT JOIN` / `RIGHT JOIN`
- `FULL OUTER JOIN`
- Autouniones (self-join)
- `UNION` y `UNION ALL`

### 03 - Funciones de agregado
- `COUNT()`, `SUM()`, `AVG()`, `MIN()`, `MAX()`
- `GROUP BY` y `HAVING`
- Subconsultas: en `WHERE`, en `FROM`, correlacionadas
- Vistas: `CREATE VIEW`
- Índices: `CREATE INDEX`
