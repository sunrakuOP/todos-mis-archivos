# 🌐 JavaScript

Apuntes, ejercicios y proyectos de JavaScript (ES6+).

---

## 📋 Temario

### 01 - Fundamentos
- Variables: `var`, `let`, `const`
- Tipos de datos: `string`, `number`, `boolean`, `null`, `undefined`, `object`
- Operadores aritméticos, de comparación y lógicos
- Condicionales: `if/else`, operador ternario, `switch`
- Bucles: `for`, `while`, `for...of`, `for...in`
- Funciones: declaración, expresión, arrow functions
- Scope, hoisting, closures

### 02 - Manipulación del DOM
- Selección de elementos: `querySelector`, `getElementById`, etc.
- Modificar contenido: `innerHTML`, `textContent`
- Modificar estilos y clases: `classList`, `style`
- Eventos: `addEventListener`, tipos de eventos
- Crear y eliminar elementos del DOM

### 03 - ES6+ (Modern JavaScript)
- Template literals
- Destructuring de arrays y objetos
- Spread / Rest operator (`...`)
- Módulos: `import` / `export`
- Clases y herencia
- `Map`, `Set`

### 04 - JavaScript Asíncrono
- Callbacks
- Promesas: `then`, `catch`, `finally`
- `async` / `await`
- `fetch` API — consumir APIs REST
- Manejo de errores asíncronos
