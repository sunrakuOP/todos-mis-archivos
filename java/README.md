# ☕ Java

Apuntes, ejercicios y proyectos de Java.

---

## 📋 Temario

### 01 - Fundamentos
- Tipos de datos primitivos: `int`, `double`, `float`, `char`, `boolean`
- Variables, constantes (`final`)
- Operadores aritméticos, relacionales y lógicos
- Condicionales: `if/else`, `switch`
- Bucles: `for`, `while`, `do-while`
- Métodos: definición, parámetros, retorno
- Arrays unidimensionales y bidimensionales

### 02 - Programación Orientada a Objetos (POO)
- Clases y objetos
- Constructor, `this`
- Getters y Setters
- Herencia: `extends`, `super`
- Polimorfismo: sobrecarga y sobreescritura (`@Override`)
- Interfaces y clases abstractas
- Encapsulación y modificadores de acceso (`public`, `private`, `protected`)

### 03 - Colecciones y generics
- `ArrayList`, `LinkedList`
- `HashMap`, `HashSet`
- Iteración con `for-each` e `Iterator`
- Generics básicos (`<T>`)
- Ordenación con `Comparator` y `Comparable`
